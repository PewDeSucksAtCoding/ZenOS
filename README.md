# This is my own personal OS creation and learning project

## Current features:
* **Multi-boot compatible**: boots with QEMU and GRUB
* **VGA-drivers**: Text printing, console clearing, and console color changing
* **Cursor**: Automatically updates as new text is printed

## How to run the OS?
1. Make sure 'i686-elf-gcc' and 'nasm' are installed on your machine
Do the following step *in this project's directory*
2. Run the 'make' command and afterwards boot the OS with the 'qemu-system-i386 -kernel myos.bin' command
